Roblox: Bank Jump Now By B Benchy

I started learning Lua programming by creating a Roblox Obby game called "Bank Jump Now By B Benchy." Players navigate obstacles to reach the finish line. I scripted several features:

Leaderstats: Tracks the player’s stage progress and saves data when they touch a SpawnLocation.

Obstruction: Includes animated obstacles: Transparency: Objects fade in and out. KillBrick: Red blocks that deplete HP to 0 upon contact. Position: Moving objects.

Sprint: Pressing LeftShift doubles the player’s speed temporarily.
